пример оформеления проекта и сборкой его с помомощью make

запуск всех целей
```shell
make
```
собрать приложение
```shell
make build
```
собрать тесты
```shell
make build_test
```
запустить приложение
```shell
make run
```
запустить тесты
```shell
make run_test
```
запустить линтер
```shell
make run_lint
```
очистить сборку
```shell
make clean
```
